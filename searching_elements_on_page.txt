Основные способы поиска элемента на странице:
1. Поиск с помощью CSS-селекторов, когда путь к элементу описывается через синтаксис CSS. Селектор — это описание пути к элементу на странице. например ниже приведены части элементов HTML-страницы, по которым можно найти элемент
	Использование потомков:	#post2 .title
	Использование дочерних элементов: #post2 > div.title
	Использование порядкового номера дочернего элемента:	#posts > .item:nth-child(2) > .title
	Использование нескольких классов:	.title.second
2. Поиск с помощью указания значений тегов или атрибутов элементов: id, tag, значение атрибута name, class
3. Поиск с помощью языка запросов XPath.
	1. XPath запрос всегда начинается с символа / или //
		el1/el2 — выбирает элементы el2, являющиеся прямыми потомками el1;
		el1//el2 — выбирает элементы el2, являющиеся потомками el1 любой степени вложенности.
	2. Символ [ ] — это команда фильтрации
		-по любому атрибуту: //img[@id='bullet']
		-по порядковому номеру: //div[@class="row"]/div[2]
		-по полному совпадению текста: //p[text()="Lenin cat"]
		-по частичному совпадению текста или атрибута: //div[contains(@class, "navbar")]
		-в фильтрации еще можно использовать булевы операции (and, or, not) и некоторые простые арифметические выражения (но вообще не стоит, наверное). Допустим, мы хотим найти картинку обязательно с data-type "animal" и именем "bullet-cat", для этого подойдет запрос: //img[@name='bullet-cat' and @data-type='animal']
	3. Символ * — команда выбора всех элементов
		-например можем найти текст в заголовке запросом //div/*[@class="jumbotron-heading"]
